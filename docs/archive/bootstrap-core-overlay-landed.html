Core/overlay split + anti-drift check — landed 2026-07-03

The core-vs-overlay boundary is now formalized and machine-guarded. A project extends a phase contract with <project>/prompts/overlay.<phase>.md, which helix/overlay.py:resolve_prompt appends to the packaged core contract — extensions extend, they never replace. A full prompts/<phase>.md replacement is a fork of the core contract: resolution ignores it and the check flags it. Self-hosting is the sanctioned identity case (the repo's own prompts/ resolve to the core files — no flag).
